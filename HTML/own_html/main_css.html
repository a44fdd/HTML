Página con estilo CSS adjuntados por la etiqueta link

Métodos de etiqueta link

  • rel: indica el tipo de rerlación recurso enlazada, si es un archivo CSS siempre se utiliza el archivo stylesheet
  • type: indica el recurso enlazado. Por defecto en los archivos CSS es "text/css"
  • href: Indica la dirección del archivo

Componentes básicos de un estilo CSS

Una regla de CSS siempre debe estar dentro de llaves {}

  • selector: Se define las etiqutas a la que se aplicara el estilo CSS
  • Declaración: Especifíca los estilos que se aplican a las etiquetas
    • Propiedad: Se indica el aspecto a modificar
    • Valor: Indica el valor modificado de la etiqueta
